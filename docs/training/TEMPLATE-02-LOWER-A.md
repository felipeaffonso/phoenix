# Template 02 — Wednesday — Lower A

Lower-body session with controlled, lumbar-aware exercise selection.

Focus:

- quadriceps
- hamstrings
- glutes
- calves
- core

Avoid:

- heavy axial loading
- aggressive hip/lumbar twisting
- heavy leg press
- heavy squat/deadlift patterns

---

## 1. Leg Extension (Machine)

**Portuguese:** Cadeira extensora

**Strong Template**

| Setting | Value |
|---|---:|
| Sets | 4 |
| Template reps | 10 |
| Target reps | 10-15 |
| Rest | 60s |

**Sticky Note**

```text
🎯 Meta: 10–15 reps

Pare quando sentir que conseguiria fazer apenas mais 2 repetições (RIR 2).

👉 Apenas a última série pode terminar em RIR 1.

Progressão:
15/15/15/15 → +5 kg

Phoenix Tips:
• Contraia o quadríceps no topo.
• Descida controlada (2–3 s).
• Não bata os pesos.
```
