# Template 01 — Tuesday — Upper A

Upper-body session focused on chest, back, lateral delts and triceps.

Design:

- stable machines/cables
- shoulder-aware pressing
- supported pulling
- no aggressive lumbar movement

---

## 1. Chest Press (Machine)

**Portuguese:** Supino na máquina / Chest press

**Strong Template**

| Setting | Value |
|---|---:|
| Sets | 3 |
| Template reps | 8 |
| Target reps | 8-12 |
| Rest | 90s |

**Sticky Note**

```text
🎯 Meta: 8–12 reps

Pare em RIR 2.

Última série:
RIR 1 se estiver sem dor no ombro.

Progressão:
12/12/12 → +2,5 a +5 kg

Phoenix Tips:
• Costas apoiadas.
• Escápulas encaixadas.
• Não tirar o ombro do banco.
```
