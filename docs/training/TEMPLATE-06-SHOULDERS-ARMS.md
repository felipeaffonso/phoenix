# Template 06 — Sunday — Shoulders + Arms

Shoulder-safe session focused on delts, biceps and triceps.

## 1. Shoulder Press (Machine)

Portuguese: Desenvolvimento máquina

Strong Template: 3 sets, 8 reps, target 8-12, rest 90s.

Sticky Note:

```text
🎯 Meta: 8–12 reps

Pare em RIR 2–3.

Progressão:
12/12/12 → +2,5 kg

Phoenix Tips:
• Pegada neutra se possível.
• Não descer além do confortável.
• Sem dor no manguito.
```
