# Template 04 — Friday — Upper B

Upper-body session focused on back, chest, biceps and rear delts.

Design:

- stable pulling
- controlled pressing
- posterior shoulder work
- no lumbar cheating

---

## 1. Iso-Lateral Row (Machine)

**Portuguese:** Remada iso-lateral máquina

**Strong Template**

| Setting | Value |
|---|---:|
| Sets | 3 |
| Template reps | 8 |
| Target reps | 8-12 |
| Rest | 90s |

**Sticky Note**

```text
🎯 Meta: 8–12 reps

Pare em RIR 2.

Progressão:
12/12/12 → +2,5 a +5 kg

Phoenix Tips:
• Peito apoiado se possível.
• Não rotacionar o tronco.
• Puxe com cotovelos.
```
