# Template 05 — Saturday — Lower B

Lower-body session focused on posterior chain, glutes, calves and core.

Design:

- posterior work without heavy deadlift
- glute work without lumbar compensation
- controlled quad stimulus
- safe core

---

## 1. Lying Leg Curl (Machine)

**Portuguese:** Mesa flexora deitada

**Phoenix Template**

| Setting | Value |
|---|---:|
| Sets | 4 |
| Template reps | 10 |
| Target reps | 10-15 |
| Rest | 75s |

**Execution Note**

```text
🎯 Meta: 10–15 reps

Pare em RIR 2.

Última série:
RIR 1.

Progressão:
15/15/15/15 → +2,5 a +5 kg

Phoenix Tips:
• Movimento controlado.
• Não levantar o quadril.
• Controle a volta.
```
