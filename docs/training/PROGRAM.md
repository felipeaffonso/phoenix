## Preferred cardio tools

1. Elliptical / Transport
2. Incline treadmill walking
3. Bike

Avoid HIIT for now.

## Safety rules

- No heavy squat.
- No heavy deadlift.
- No very heavy leg press.
- No aggressive twisting near hip/lumbar region.
- Machines and supported patterns are preferred.
- Shoulder work must stay pain-free.
- Lumbar discomfort overrides the plan.

## Strong template principle

Each Strong exercise receives:

- Strong exercise name
- minimum template reps
- rest time
- Sticky Note with real rep range, progression and execution tips
